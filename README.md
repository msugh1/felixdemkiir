# Felix Demkiir — Cybersecurity & Cloud Security Portfolio

A highly polished, interactive portfolio and lab notebook highlighting cybersecurity research, cloud infrastructure configurations, threat detection, incident response, and web applications.

🔗 **GitHub Repository:** [github.com/msugh1/portfoilio](https://github.com/msugh1/portfoilio)

---

## 🛠️ Technology Stack

- **Framework & Routing:** React 18, Vite, React Router
- **Styling:** Tailwind CSS (v4.0) with deep HSL dark modes, customized animations, and responsive flex grids
- **Icons:** Lucide React
- **Data Visualization:** Recharts
- **Components:** Radix UI primitives & custom styled components

---

## ✨ Features & Sections

- **Hero & Profile:** A beautifully integrated entry section showcasing profile cards, professional credentials, and a dark theme (`#0a1124`) blended using layered linear gradients.
- **Featured Projects:**
  - **Secure Cloud Deployment:** Decentralized multi-tier e-commerce platform running PrestaShop on secure AWS EC2 & RDS MySQL.
  - **BloodLink:** Hospital blood bank tracking with real-time stock notifications and forecasting charts.
  - **What's Next:** AI-powered personalized career roadmap generator using Google Gemini.
- **Findings & Lab Notebook:** Detailed documentation and writeups of cybersecurity coursework and hands-on laboratory exercises:
  - Enterprise secure network design for **KafiTech Solutions** in Cisco Packet Tracer (VLANs, ACLs, DMZ segmentation).
  - Authorized exploitation and vulnerability assessment of a local **Metasploitable 2** target.
  - Web application penetration testing on **DVWA** using OWASP ZAP scanners and custom manual payloads.
  - Traffic analysis and packet analysis using **Wireshark** display filters.
  - Investigative analysis of malware-infected system logs.
- **Interactive Tooltips & Modal Certifications:** Smooth popups rendering certifications like SC-900, 3MTT Cybersecurity, Google Cybersecurity, and CPPS.

## 🚀 Running the Project Locally

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (v18 or higher) and `npm` installed.

### 1. Install Dependencies

Clone the repository and run:
```bash
npm install
```

### 2. Run the Development Server

Start the local server with hot module reloading (HMR):
```bash
npm run dev
```
Open **`http://localhost:5173`** in your browser to view the application.

### 3. Build for Production

Bundle and optimize all static assets (HTML, CSS, JS, PDF lab documents) for production:
```bash
npm run build
```
The compiled output will be generated inside the `/dist` directory, fully ready for static hosting.
